## ✨ Sobre o projeto

A **Petalyn** é um site desenvolvido para apresentar um catálogo elegante e acolhedor de buquês e arranjos florais.

O objetivo do projeto é oferecer uma experiência visual delicada, moderna e encantadora, permitindo que os visitantes encontrem o arranjo perfeito para cada ocasião especial.

---

## 🌷 Funcionalidades

- Página inicial com apresentação da marca
- Hero section com identidade visual exclusiva
- Catálogo de buquês
- Slider interativo com destaques
- Seção de ocasiões especiais
- Layout responsivo para desktop e dispositivos móveis
- Animações suaves utilizando AOS
- Navegação intuitiva

## 🚀 Como executar

1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/petalyn.git
```

2. Abra a pasta do projeto

```bash
cd petalyn
```

3. Execute o arquivo:

```bash
index.html
```

ou utilize a extensão **Live Server** do VS Code.
