# 🌸 Petalyn

Uma floricultura virtual desenvolvida para apresentar buquês e arranjos de forma delicada, moderna e interativa.

## ✨ Sobre o projeto

A Petalyn é um projeto de site para floricultura criado com foco em uma experiência visual agradável, utilizando animações, design responsivo e uma identidade visual romântica e elegante.

O objetivo é permitir que clientes explorem catálogos de flores e encontrem o arranjo ideal para cada ocasião especial.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Swiper.js
- AOS (Animate On Scroll)
- Font Awesome
